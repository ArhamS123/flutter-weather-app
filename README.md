# 🌤 Weather App (Flutter)

A modern weather application built with Flutter.

## Features
- Live weather data
- Search any city
- Offline caching (Hive)
- Clean UI with glass design
- Forecast display

## Tech Stack
- Flutter
- Provider
- REST API (WeatherAPI)
- Hive (Offline storage)


## API Used
WeatherAPI: https://www.weatherapi.com/
